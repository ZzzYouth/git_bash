2. $ cd ~ | Открыть домашнюю директорию через терминал

3. $ pwd  | Определить имя папки, в которой вы находитесь

4. $ mkdir test1 | Создать внутри этой папки каталог с именем test1

5. $ cd test1    | Перейти в папку test1

6. $ touch file1.txt file2.txt file3.txt | Создать файл 1,2 и 3 внутри каталога test1

7. $ ls | Проверить содержимое каталога test1

8. $ cd | Перейти в домашнюю директорию

9. $ mkdir test2 | Создать папку test2 внутри домашней директории

10. $ rmdir test2 | Удалить папку test2

11. $ rm ~/test1/file2.txt| Удалить файл 2 из папки test1

12. Создать папку в домашней директории test3 и добавить в нее два файла

$ cd
$ mkdir test3
$ cd test3
$ touch test1.txt test2.txt 

13. Удалить папку test3

$ cd
$ rm -r ~/test3 

14. $ mkdir test4 | Создать папку test4 в домашней директории

15. $ ~/test1/file1.txt ~/test1/file3.txt ~/test4/ | Переместить файлы 1 и 3 из папки test1 в папку test4

16. $ echo -e "line 1\nline 2\nline 3" >> ~/test4/file1.txt | Добавить в файл 1 три строки со словами line

17. $ cat ~/test4/file1.txt | Посмотреть содержимое файла 1

18. $ echo -e "line 1\nline 2\nline 3" >> ~/test4/file3.txt | Добавьте в файл 3 три строки со словами line

19. $ cat ~/test4/file1.txt ~/test4/file3.txt | Просмотрите содержимое двух файлов (1 и 3) сразу

20. $ echo -e "новая строка 1\nновая строка 2\nновая строка 3" > ~/test4/file1.txt | Используя один из редакторов замените все строки в файле 1
